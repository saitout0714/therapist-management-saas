THERAPIST - セラピスト一覧ページ

SCHEDULE - 出勤予定スケジュール

CAST PROFILE - セラピスト個人スケジュール

WordPressの各キャスト紹介ページ（例: https://kokoro-rinse.com/cast/きたがわ/）に埋め込んだ場合の表示イメージです。(data-mode="single-therapist")

COMBINED - セラピスト＆スケジュール複合表示

1つのページに両方をタブで切り替えて表示させたい場合（例：トップページなど）の表示イメージです。(data-mode="all")

WordPressへの導入貼り付けコード

WordPressの固定ページ（ブロックエディタ）で「カスタムHTML」ブロックを作成し、それぞれ表示させたいページに応じて以下のタグを貼り付けてください。

1. セラピスト一覧ページ (therapist) に貼り付けるコード

<!-- data-cast-url-pattern には、写真・名前をクリックした際の個人ページURL規則を指定します（デフォルト: /cast/{name}/） --> <div id="yoyakl-widget" data-shop-code="demo-shop" data-mode="therapists" data-cast-url-pattern="/cast/{name}/"></div> <script src="loader-url" defer></script>

2. スケジュールページ (schedule) に貼り付けるコード

<div id="yoyakl-widget" data-shop-code="demo-shop" data-mode="schedule" data-cast-url-pattern="/cast/{name}/"></div> <script src="loader-url" defer></script>

3. セラピスト個人紹介ページ (例：きたがわ様のページなど) に貼り付けるコード

<div id="yoyakl-widget" data-shop-code="demo-shop" data-mode="single-therapist" data-therapist-name="きたがわ"></div> <script src="loader-url" defer></script>

デザインの上書き調整 (CSSのカスタマイズ)

セージグリーン（#758e7b）がデフォルトに設定されていますが、WordPress側のCSSでさらに微調整が可能です。

/* WordPressの「外観 > 追加CSS」等に追記することで配色を自由に変更できます */ #yoyakl-widget { --yk-pink: #6E8272; /* メインカラー */ --yk-pink-hover: #566759; /* ホバー時のカラー */ --yk-pink-light: #f3f6f4; /* 薄い背景色 */ }
